This was my first time using html and css, with this primarily being learned in Web Typography class.

## Fonts

Quokka, Argent Pixel 

## Process

My main idea going into this was to expand on the theming of The Pipeworks, and what little I had made so far. The mockups I made were in figma, where I tested out the different colors and shape language I would use. I chose fonts like Quokka because I wanted to lean into a hybrid of modern and also early web styling, where the web pages would sometimes represent a location that you could click around. So when I placed everything in its supposed place and marked up what different assets would be in the code, it looked very illustrative. Coding itself was mostly just setting up all the assets correctly. Getting the right size, hover effects, links, custom cursor, etc. The grid making was at the end, that way I could see all the objects in relation to one another, and I would say it looks pretty good.
